# SFLCB: A Single-Loop First-Order Algorithm for Linearly Constrained Bilevel Optimization

This repository provides the code for the experiments in the NeurIPS 2025 paper, “A Single-Loop First-Order Algorithm for Linearly Constrained Bilevel Optimization.”

We adapt and modify the code from https://github.com/Liuyuan999/Penalty_Based_Lagrangian_Bilevel [1].

Reference:
[1]: Liuyuan Jiang, Quan Xiao, Victor M. Tenorio, Fernando Real-Rojas, Antonio Marques, Tianyi Chen. A Primal-Dual-Assisted Penalty Approach to Bilevel Optimization with Coupled Constraints. NeurIPS 2024.
